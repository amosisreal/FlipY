# FlipY

A small native Windows app that toggles Y-axis inversion for mouse movement.

## Features
- Big toggle switch in the main window
- System tray icon with click-to-toggle and exit menu
- Global hotkey: Ctrl+Alt+Y

## Build
This project uses .NET 6 WinForms. Install the .NET 6 SDK, then run:

```powershell
dotnet build

dotnet run
```
